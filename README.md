# OBS Lite Recorder

OBS Lite Recorder is an Android app for recording rides with an **[OpenBikeSensor Lite](https://www.openbikesensor.org/docs/lite/)** and uploading them to an OpenBikeSensor portal.

Code used in this app originates from:
- https://github.com/openbikesensor
- https://github.com/simra-project/simra-android/tree/master

The logo was created by Lukas Betzler:  
https://github.com/pnfzygrzgf-svg/OpenBikeSensor-Logo

This app was created using *vibecoding*. **Don't trust, verify!**

## Installation (Android)

This app is **not** distributed via the Play Store, but as an **APK** via GitHub.

### 1) Download the APK

1. Open the project's GitHub page:  
   https://github.com/pnfzygrzgf-svg/ObsLiteRecorder
2. Go to **Releases**:  
   https://github.com/pnfzygrzgf-svg/ObsLiteRecorder/releases
3. Download the latest **.apk** from the release assets.

### 2) Allow installation from unknown sources

When you open the APK, Android may initially block the installation.

- Tap **Settings** / **More info** in the dialog
- Enable **"Allow apps from this source"** (for the browser or file manager you used to open the APK)

> Note: For **debug.apk**, Android/Play Protect may show a warning. This is normal for APKs installed outside the Play Store.

### 3) Install the APK

- Open the downloaded APK (Downloads / File Manager)
- Tap **Install**
- Launch the app
