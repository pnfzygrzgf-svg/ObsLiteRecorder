# ObsLiteRecorder

ObsLiteRecorder ist eine Android-App zum Aufzeichnen von Fahrten mit einem **OpenBikeSensor Lite** und zum Hochladen der bin-Dateien mit den Informationen zur Fahrt in ein **OpenBikeSensor-Portal.

Code, der in dieser App verwendet wird, stammt von:
https://github.com/openbikesensor
https://github.com/simra-project/simra-android/tree/master
