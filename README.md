# OBS Lite Recorder

OBS Lite Recorder ist eine Android-App zum Aufzeichnen von Fahrten mit einem **[OpenBikeSensor Lite](https://www.openbikesensor.org/docs/lite/)** und zum Hochladen dieser in ein OpenBikeSensor-Portal.

Code, der in dieser App verwendet wird, stammt von:

- https://github.com/openbikesensor  
- https://github.com/simra-project/simra-android/tree/master

Das verwendete Logo wurde erstellt durch Lukas Betzler:  
https://github.com/pnfzygrzgf-svg/OpenBikeSensor-Logo

Die App ist mittels *Vibecoding* erstellt worden. **Don't trust verify!**


## Installation (Android)

Diese App wird **nicht** über den Play Store verteilt, sondern als **APK** über GitHub.


## Installation direkt auf dem Smartphone (APK)

### 1) APK herunterladen
1. Öffne die GitHub-Seite des Projekts:
   - Repo: https://github.com/pnfzygrzgf-svg/ObsLiteRecorder
2. Gehe zu **Releases**:
   - Releases: https://github.com/pnfzygrzgf-svg/ObsLiteRecorder/releases
3. Lade die neueste **.apk** aus den Release-Assets herunter.

### 2) „Unbekannte Apps installieren“ erlauben
Wenn du die APK öffnest, blockiert Android die Installation oft zuerst.

- Tippe im Dialog auf **Einstellungen** / **Mehr Informationen**
- Erlaube **„Apps aus dieser Quelle installieren“** (für den Browser oder Dateimanager, mit dem du die APK öffnest)

> Hinweis: Bei **debug.apk** kann Android/Play Protect warnen. Das ist bei APKs außerhalb des Play Store üblich.

### 3) APK installieren
- Öffne die heruntergeladene APK (Downloads/Dateimanager)
- Tippe **Installieren**
- Starte die App
